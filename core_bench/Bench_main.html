Simple example of using bench: { let main () = let t1 = Bench.Test.create ~name:"ArrayCreateInt1" (fun () -> ignore (Array.create ~len:200 0)) in let t2 = Bench.Test.create ~name:"ArrayCreateInt2" (fun () -> ignore (Array.create ~len:300 0)) in Command.run (Bench.make_command [t1; t2;]) let () = main () }
For default values of all the optional arguments, see the module called Defaults in bench_main.ml.
For some notes on the design space of bench and TODO items, see: http://docs/programming/performance/benchmarking.html
